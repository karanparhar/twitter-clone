# twitter-clone

This is sample twitter-clone

## Quick start

### Prerequisites
- [go](https://golang.org/dl/) version v1.10+

### Steps to run

```
$ mkdir $GOPATH/src/github.com/
$ cd $GOPATH/src/github.com/
$ git clone https://karanjit@bitbucket.org/karanjit/twitter-clone.git
$ cd twitter-clone
$ make docker
```
